# Gesture-Controlled-Arcade-Game---SoC-26

A playable game that responds to your hand gestures live, with an optional HUD overlay showing the detected gesture and a quick calibration mode at startup.
 
So far I've built a Gesture recogniser using Python, OpenCV, and MediaPipe. Weeks 1–2 cover environment setup and MediaPipe fundamentals. Week 3 implements live hand tracking, extracting 21 landmark coordinates per frame to detect gestures like fist, peace sign, and thumbs up. Week 4 engineers a standalone gestures.py module using normalized landmark distances for camera-distance-invariant classification of 6 gestures and fine tuning the parameters for eliable, stable recognition across varying lighting conditions and hand-to-camera distances.
